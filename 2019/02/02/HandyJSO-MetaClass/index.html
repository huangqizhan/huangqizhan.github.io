HandyJSON 元数据 MetaData

在程序运行起来之后每个类的信息都会加载到内存中，而类的信息是以结构体的形式存在， HandyJSON中MetaData.Class主要存储类的基信息 ，MetaData.Struct 主要存储结构体的信息
